# Hadoop commands

## Usage

```bash
$ hdfs dfs [GENERIC_OPTIONS] [COMMAND_OPTIONS]
```

## Examples

```bash
$ hdfs dfs -ls /nyc_taxi
$ hdfs dfs -du -h /nyc_taxi
$ hdfs dfs -rm -r /nyc_taxi
```
